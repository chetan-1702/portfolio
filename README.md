# Dissertations

## MSc — [A Complexity-Based Analysis of Deepfake Detection Mechanisms](./msc-deepfake-detection)
MSc Cyber Security & Pen Testing, Middlesex University London, 2024–25.
Compares a CNN (XceptionNet) against a CNN-LSTM hybrid for deepfake detection on FaceForensics++ and Celeb-DF-v2, evaluating both detection accuracy and computational cost (training time, memory, FLOPs).

## BSc — [Analysis of VPN Encryption Techniques, Vulnerabilities and Behaviours in Cloud and On-Premise Environments](./bsc-vpn-analysis)
BSc (Hons) Cyber Security & Digital Forensics, Middlesex University Mauritius, 2024.
A meta-analysis of VPN tunneling protocols (GRE, IPSec, L2TP), their encryption/authentication mechanisms, common attack vectors (DoS, MiTM, data spoofing), and the trade-offs between traditional on-premise and cloud-based VPN deployments.
